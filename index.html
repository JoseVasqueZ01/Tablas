Diferencias entre Git Local y Git Remoto

Característica Git Local Git Remoto
Operación Se ejecuta en la máquina del usuario, sin necesidad de conexión a internet Opera en un servidor, generalmente en la nube, y requiere conexión a internet
Colaboración Ideal para uso individual o pequeños equipos Facilita la colaboración entre múltiples desarrolladores, permitiendo compartir y actualizar proyectos fácilmente
Acceso Los repositorios son privados por defecto en el entorno local Los repositorios pueden ser públicos o privados, y se pueden compartir en la web a través de plataformas como GitHub
Gestión de ramas Los usuarios pueden crear y fusionar ramas localmente Permite la creación de ramas y la fusión de cambios a través de la plataforma web
Seguridad La seguridad depende del entorno local y de las políticas de seguridad implementadas La seguridad es manejada por la plataforma web y puede incluir características como autenticación y permisos de acceso


Comandos basicos de Git remoto y Git local

Categoría Comando Descripción Ejemplo de uso
Git Local git init Inicializa un nuevo repositorio Git en el directorio actual git init
git add Agrega cambios en el directorio de trabajo al área de preparación git add archivo.txt
git commit Guarda los cambios del área de preparación en el repositorio git commit -m "Mensaje descriptivo"
git status Muestra el estado del repositorio, incluyendo cambios que se han añadido al área de preparación git status
git log Muestra el historial de confirmaciones git log
git diff Muestra las diferencias entre los cambios hechos y los que están en el repositorio git diff
Git Remoto git clone Clona un repositorio remoto en un directorio local git clone https://github.com/usuario/repositorio.git
git fetch Obtiene los cambios del repositorio remoto git fetch origin
git pull Obtiene cambios del repositorio remoto y los fusiona con el repositorio local git pull origin master
git push Envía los cambios del repositorio local al repositorio remoto git push origin master
git remote Administra los repositorios remotos git remote add origin https://github.com/usuario/repositorio.git
git remote -v Muestra los repositorios remotos asociados con el repositorio local git remote -v
